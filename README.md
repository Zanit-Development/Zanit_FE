### commit

```powershell
git commit -m "(#issue_number) Feat : 팔로우 버튼 기능 구현 "
```

```
feat : 새로운 기능과 관련된 것을 의미한다.
fix : 오류와 같은 것을 수정했을 때 사용한다.
test : test를 추가하거나 수정했을 때를 의미한다.
docs : 문서와 관련하여 수정한 부분이 있을 때 사용한다 (주석포함)
style : 코드의 변화와 관련없는 포맷이나 세미콜론을 놓친 것과 같은 부분들을 의미한다.
design :css 등 사용자 ui 관련 디자인변경
refactor : 코드의 리팩토링을 의미한다.
build : 빌드 관련 파일 수정에 대한 커밋
chore : 변수 이름 변경이나 패키지 매니저 설정 등 여러 production code와 무관한 부분들을 의미한다.
remove : 파일 삭제 작업
rename : 파일이나 폴더명 수정 및 이동 작업
BREAKING CHANGE : API 변경
```
